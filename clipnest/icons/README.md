# ClipNest 图标说明

此目录包含插件的图标文件，用于不同尺寸的显示。

## 文件列表

- `icon16.png` - 16x16 像素图标，用于浏览器地址栏
- `icon48.png` - 48x48 像素图标，用于扩展管理页面
- `icon128.png` - 128x128 像素图标，用于扩展商店和安装提示

## 设计要求

图标应符合以下设计规范：

1. **风格统一**：采用现代扁平化设计风格
2. **颜色**：以橙色 (#FF6B35) 和白色为主，与插件主色调一致
3. **清晰度**：确保在小尺寸下仍能清晰识别
4. **格式**：PNG 格式，透明背景

## 占位图替换

当前为占位图标，实际使用时请替换为符合设计规范的图标文件。
